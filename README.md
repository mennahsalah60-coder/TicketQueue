"# TicketQueue" 

Queue Management System: 

This is a simple JavaScript queue application that allows users to:

Add names to a waiting list:

  Serve people one by one
  Display who is currently being served
  Show counters for waiting and served

Features:

  Feature	Description
  Add to Queue	User enters a name and adds it
  Serve Next	Removes the first person and moves them to "Now Serving"
  Live Count	Shows number of people waiting and served
  Dynamic UI	Updates list instantly
  Status Label	Adds a word/emoji behind the served names

Technologies Used:

  HTML
  CSS / Bootstrap 
  JavaScript
